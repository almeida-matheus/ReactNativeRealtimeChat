# React-native Real-time chat

A little Real-time chat app (It's just for learning)

# Relevant libs that i'm using
  - react-native

### Installation
Install the dependencies and start the server.

```sh
$ npm install
$ npm start
$ react-native run-android or react-native run-ios
```
